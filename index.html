django-grappelli

A jazzy skin for the Django admin interface.

The Django admin interface is a simple yet highly customizable app in order to manage your websites data — some might call it a CMS. On top of Djangos admin interface, Grappelli gives you a consistent and grid-based look & feel and adds some nifty features.

Documentation on RTD

Code on GitHub

Package on PyPI

 

 

 

 

  • Consistent and grid-based look & feel
  • Related lookups with Autocompletes
  • Inline sortables with Drag & Drop
  • Compass-based CSS (coming with 2.4)
  • Integrated TinyMCE theme
  • Customizable dashboard
  • Improved collabsibles
  • Unified jQuery-Plugins
  • Compatible with django—filebrowser

The latest stable version is 2.3.5, released in October 2011. Grappelli requires Django 1.3.x (official release) and is licensed under the New BSD License.
